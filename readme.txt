Displays your most recent post as a featured post, on only the first page of your Wordpress blog

== Description ==

This plugin allows you to have different styling for your newest post on Wordpress. Features CSS classes right in the plugin options page, for easy customization. <br /><br />This plugin was created by <a href="http://www.bluefountainmedia.com/">Blue Fountain Media</a>, a <a href="http://www.bluefountainmedia.com/website-design-development">website development</a> and marketing company in New York City.<br /><br />You can find more information regarding the <a href="http://www.bluefountainmedia.com/blog/latest-post-plugin" >Latest Post Plugin</a> here: <a href="http://www.bluefountainmedia.com/blog/latest-post-plugin">http://www.bluefountainmedia.com/blog/latest-post-plugin</a>

== Installation ==

1. Upload the entire folder to your /wp-content/plugins/ folder.
2. Go to your plugins page and activate plugin.
3. Go to Settings>Latest Post and choose your settings. 
4. Add <?php if(function_exists('latestPost')) latestPost(); ?> anywhere outside the loop.
5. Edit plugins/latestPost/style.css to style your latest post.
